# .NET Conference 2025 - Blazor e .NET 10: cosa c'è di nuovo?

This is the talk (Italian language) I did at [.NET Conference 2025 Milan](https://dotnetconference.it/e/sessione/3788/Blazor-in-NET-10-cosa-c%E2%80%99e-di-nuovo).

## Requirements

- .NET 10 SDK
- Aspire 13.0

## Projects

The **BlazorWasm/BlazorNet10.WasmApp** contains a standalone Blazor WebAssembly application which has been used to show the diagnostic and profiling features of Blazor with .NET 10.

Tha ***BlazorNet10.AppHost** is the Aspire AppHost project to run in order to view the features of Blazor Server and SSR, such as metrics, traces, validation enhancement and persistence component state.
